macOS Cheatsheet for Hackintosh (Windows Keyboard) ⌨️🍏

Welcome to macOS! Since you are using a standard Windows PC keyboard on your Hackintosh, the physical keys map slightly differently to macOS. Here is a comprehensive guide to mastering your new system without getting confused!

🔑 The Magic Key: Windows = Command (⌘)

In macOS, the primary shortcut key is the Command (⌘) key. On your physical keyboard, this is the Windows key. Any time a Mac tutorial tells you to press "Command", you should press the "Windows" key.
  • Alt Key = Acts as the Option (⌥) key on Mac.
  • Ctrl Key = Acts as the Control (⌃) key on Mac (mostly used to mimic a right-click).

👶 Absolute Beginner Basics

If this is your first time using a Mac, here is how to do the absolute essentials:

ActionHow to do it
Open Terminal (or any App)Press Windows + Space to open Spotlight Search, type Terminal (or any app name/math problem), and hit Enter.
Right-ClickTap the trackpad with two fingers, OR hold the Ctrl key while clicking.
Change Volume & BrightnessPress F11/F12 for Volume and F1/F2 for Brightness. You can also click the Control Center icon (two toggles) in the top right menu bar.
Close vs. QuitClicking the red "X" only closes the window but leaves the app running. To free up memory and fully close it, press Windows + Q.

💡 Essential Mac & Hackintosh Tips

  • Installing Apps (.dmg vs .exe): Mac apps usually download as .dmg (Disk Image) files. When you open one, a small window pops up showing the App icon and an Applications folder. Do not just double-click the app! You must click and drag the app icon into that Applications folder.
  • Fixing "Backwards" Scrolling: By default, macOS uses "Natural Scrolling" (like a smartphone). If you are using a standard mouse wheel and hate this, go to System Settings -> Mouse and turn off Natural Scrolling.
  • ⚠️ The Hackintosh Golden Rule: Because you are running macOS on PC hardware, you cannot just click "Update Now" for major new macOS versions. You must update your OpenCore EFI folder and kexts first. Always check Hackintosh forums before updating!

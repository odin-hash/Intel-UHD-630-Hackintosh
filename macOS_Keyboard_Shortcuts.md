# macOS Cheatsheet for Hackintosh (Windows Keyboard) ⌨️🍏

Welcome to macOS! Since you are using a standard Windows PC keyboard on your Hackintosh, the physical keys map slightly differently to macOS. Here is a comprehensive guide to mastering your new system without getting confused!

> [!IMPORTANT]
> **🔑 The Magic Key: Windows = Command (⌘)**
> In macOS, the primary shortcut key is the **Command (⌘)** key. On your physical keyboard, this is the **Windows** key. Any time a Mac tutorial tells you to press "Command", you should press the "Windows" key.
> 
> * **Alt Key** = Acts as the **Option (⌥)** key on Mac.
> * **Ctrl Key** = Acts as the **Control (⌃)** key on Mac (mostly used to mimic a right-click).

## 👶 Absolute Beginner Basics

If this is your first time using a Mac, here is how to do the absolute essentials:

| Action | How to do it |
| :--- | :--- |
| **Open Terminal (or any App)** | Press `Windows + Space` to open Spotlight Search, type `Terminal` (or any app name/math problem), and hit `Enter`. |
| **Right-Click** | Tap the trackpad with **two fingers**, OR hold the `Ctrl` key while clicking. |
| **Change Volume & Brightness** | Press `F11/F12` for Volume and `F1/F2` for Brightness. You can also click the Control Center icon (two toggles) in the top right menu bar. |
| **Close vs. Quit** | Clicking the red "X" only closes the window but leaves the app running. To free up memory and fully close it, press `Windows + Q`. |

## 💡 Essential Mac & Hackintosh Tips

* **Installing Apps (.dmg vs .exe):** Mac apps usually download as `.dmg` (Disk Image) files. When you open one, a small window pops up showing the App icon and an Applications folder. **Do not just double-click the app!** You must *click and drag* the app icon into that Applications folder.
* **Fixing "Backwards" Scrolling:** By default, macOS uses "Natural Scrolling" (like a smartphone). If you are using a standard mouse wheel and hate this, go to **System Settings -> Mouse** and turn off **Natural Scrolling**.
* **⚠️ The Hackintosh Golden Rule:** Because you are running macOS on PC hardware, you cannot just click "Update Now" for major new macOS versions. You *must* update your OpenCore EFI folder and kexts first. Always check Hackintosh forums before updating!

## 📝 Text Editing & Everyday Use

| Action | PC Keyboard Shortcut |
| :--- | :--- |
| **Copy** | `Windows + C` |
| **Paste** | `Windows + V` |
| **Cut** | `Windows + X` |
| **Undo** | `Windows + Z` |
| **Redo** | `Windows + Shift + Z` |
| **Select All** | `Windows + A` |
| **Save** | `Windows + S` |
| **Find** | `Windows + F` |

## 📁 Finder & File Management

| Action | PC Keyboard Shortcut | Description |
| :--- | :--- | :--- |
| **Open a File** | `Windows + O` or `Windows + ↓` | (Pressing just `Enter` like on Windows will *rename* it). |
| **Rename a File** | `Enter` | Renames the selected file/folder. |
| **Preview File** | `Spacebar` | Opens Quick Look to preview a file instantly. |
| **Select Multiple Files** | `Windows + Click` | Use the Windows key instead of Ctrl to select individual files. |
| **Delete File (to Trash)** | `Windows + Backspace` | Moves the selected item to the Trash. |
| **Empty Trash** | `Windows + Shift + Backspace` | Empties the Trash. |
| **New Folder** | `Windows + Shift + N` | Creates a new folder. |
| **Go to Folder (Paths)** | `Windows + Shift + G` | **Essential!** Lets you type a path like `/Volumes/EFI` to access hidden drives. |
| **Show Hidden Files** | `Windows + Shift + .` | Toggles visibility of hidden files (files starting with a dot). |

## 🪟 Window & App Management

| Action | PC Keyboard Shortcut | Description |
| :--- | :--- | :--- |
| **Close Window** | `Windows + W` | Closes the frontmost window. |
| **Quit App** | `Windows + Q` | Completely quits the active application. |
| **Switch Apps** | `Windows + Tab` | Switches to the next most recently used app (like `Alt + Tab`). |
| **Hide App** | `Windows + H` | Hides the windows of the front app. |
| **Show Desktop** | `Windows + F3` | Temporarily pushes all windows aside to show your desktop. |

## 📸 Screenshots & Screen Recording

| Action | PC Keyboard Shortcut | Description |
| :--- | :--- | :--- |
| **Screenshot (Full)** | `Windows + Shift + 3` | Captures the entire screen. |
| **Screenshot (Selection)**| `Windows + Shift + 4` | Turns cursor into a crosshair to select an area. |
| **Screenshot (Window)** | `Windows + Shift + 4`, then `Space` | Captures a specific window with a nice drop shadow. |
| **Screen Recording** | `Windows + Shift + 5` | Opens the Screenshot utility to record video. |

## 🔧 System & Power

| Action | PC Keyboard Shortcut | Description |
| :--- | :--- | :--- |
| **Force Quit** | `Windows + Alt + Esc` | Opens the Force Quit window (similar to `Ctrl + Alt + Delete`). |
| **Lock Screen** | `Ctrl + Windows + Q` | Immediately locks your Mac screen. |
